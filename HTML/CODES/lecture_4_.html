Block Elements

Block elements typically start on a new line and take up the full width available by default. They are used for structuring and organizing content.

Heading 1: Lorem ipsum dolor sit amet.

The <h1> to <h6> tags are used for headings, where <h1> is the highest and <h6> is the lowest level.

This is a paragraph. The <p> tag is used to define blocks of text. It automatically adds space before and after the text.

This is a division. The <div> tag is used to group content into sections for styling or layout purposes.

An unordered list is created using the <ul> tag, with list items defined using the <li> tag:

  • Item 1: Lorem ipsum dolor sit amet.
  • Item 2: Lorem ipsum dolor sit amet.
  • Item 3: Lorem ipsum dolor sit amet.

An ordered list is created using the <ol> tag, with items in a numbered sequence:

  1. Step 1: Lorem ipsum dolor sit amet.
  2. Step 2: Lorem ipsum dolor sit amet.
  3. Step 3: Lorem ipsum dolor sit amet.

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. They are used for text styling or embedding other inline content.

This is bold text. The <b> tag makes the text bold but does not add semantic importance.

This is italic text. The <i> tag styles the text in italics for stylistic or alternate voice purposes.

The <span> tag is an inline container for text or other inline elements. It is commonly used for styling or JavaScript manipulations:
